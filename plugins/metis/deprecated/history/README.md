# History

Design-process documents from Metis's v0.1 build. Preserved as historical record, not as current documentation.

For current docs, see the project [`README.md`](../../README.md) at the repo root.

## Files

- **`metis-v0.1-handoff.md`** — the original design handoff that produced Metis v0.1. Captures the workflow, principles, command set, and the long string of refinements (1–15) that landed during the build. Some path conventions in this doc (`metis-<name>/` directory prefix, teaching-register skill layer) reflect intermediate states that were superseded by Refinements 12 and 14; the Refinement narratives in the doc itself record the transitions.
- **`metis-v0.1-skills-review.md`** — pre-ship holistic review of the skills layer. References "fifteen skills" and `template/.claude/skills/metis/` paths from before the plugin restructure; preserved for the review patterns it captures, not as a current count or layout.

These documents are not loaded by any skill or subagent at runtime.
